# Games For Murkmod
This is a collection of games for murkmod

# How to install
To install this, you must follow these steps!

1. Open crosh (or VT2) and then open the root shell

2. Now that you are signed in as root, type this:
   ```
   cd mnt/stateful_partition/murkmod/plugins
   ```
3. After that, you could type `ls` if you want or whatever shit, but that doesn't matter, cause now we are going to install this shit!

4. Go ahead and type the raw user content. It should look like this:
   ```
   curl -O https://raw.githubusercontent.com/REST/OF/SHIT
   ```
5. THERE YOU GO! You did it, you installed this shit, good job!
